# Personal Vault

A simple Solidity smart contract that allows the owner to store ETH with a time lock. Funds can only be withdrawn after the unlock time has passed.

## Features

- Deposit ETH into the vault
- Withdraw ETH after the unlock time
- Extend the lock period
- Owner-only withdrawal

## Technologies

- Solidity ^0.8.20
- Remix IDE
- MetaMask
- Sepolia Testnet
- Etherscan
